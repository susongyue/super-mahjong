## 角色详情展开动画（game.html + battle_round.html）

重做角色详情界面，点击卡片后从卡片位置平滑过渡展开为居中详情面板，关闭时反向收缩回原卡片位置。

**实现方式**：GSAP 手动 FLIP 动画

1. **事件绑定**：点击时传递被点击的卡片 DOM 元素给 `openFullscreen(idx, cardEl)`
2. **openFullscreen 展开**（~0.45s）：
   - 获取卡片 `getBoundingClientRect()`
   - `gsap.set` overlay 初始位置/大小与卡片完全重合
   - 初始背景色设为卡片背景色（防止颜色跳变）
   - 内容元素 opacity 初始为 0
   - `gsap.to` 动画到居中目标位置（`top: 5vh`, `left: 居中`, `width: min(900px, 92vw)`）
   - 背景色同步过渡到 `var(--canvas)`
   - 内容元素在动画后半段 stagger 淡入
3. **closeFullscreen 收缩**（~0.35s）：
   - 内容先淡出（0.15s）
   - overlay 收缩回原卡片位置/大小
   - 完成后 `clearProps: 'all'` 清理 inline 样式并隐藏
4. **防重复点击**：`fsAnimating` 标志位阻止动画过程中再次触发

涉及文件: `public/game.html`, `public/battle_round.html`

## 修复详情弹窗闪烁问题

详情弹窗展开前一瞬间会闪一下完整大小的 overlay，再缩回卡片位置展开。根因是 `.char-fullscreen-overlay` 移除 `.hidden`（`display: none`）后，在 GSAP 设置初始位置前有一帧以默认 CSS 完整尺寸渲染。

**修复**：给 `.char-fullscreen-overlay` 和 `.char-fullscreen-backdrop` 默认添加 `opacity: 0; visibility: hidden;`，确保移除 `.hidden` 后仍然不可见，直到 GSAP 动画接管显示。

涉及文件: `public/style.css`
